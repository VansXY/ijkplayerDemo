### 一、下载ijkplayer

在一个合适的位置新建一个文件夹, 假设为桌面, 文件夹名为 **ijkplayer**

```
# 进入到刚刚新建的文件夹内
cd ~/Desktop/ijkplayer/

# 获取ijkplayer源码
git clone https://github.com/Bilibili/ijkplayer.git ijkplayer-ios

# 进入源码目录
cd ijkplayer-ios

```
